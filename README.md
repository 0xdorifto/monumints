# Monumints 🏰

Monumints is a collectible NFT album for UNESCO world heritage sites.

## Usage

Install dependencies:

```bash
npm i
```

To setup the database, after creating your [planetscale repo](https://planetscale.com/) and following the instructions;

Create tables:

```bash
npx prisma db push
```

Seed the database:

```bash
npx prisma db seed
```

Run the app:

```bash
npm run dev
```
